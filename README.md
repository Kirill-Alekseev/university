# Foxminded University

###*Application is made to manage student's and teacher's timetable. Also, it allows editing university data.*  

####Technologies: 
* Java 8
* Maven
* PostgreSQL, H2
* Spring Boot, IoC, MVC, Data
* REST
* Thymeleaf, Bootstrap

To run the application:
```
mvn spring-boot:run
```

**Required free port 8080**
